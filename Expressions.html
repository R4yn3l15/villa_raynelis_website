Expressions (Closer)

Interactive Visuals

Real-time generative visuals driven by facial pose recognition.

Concept: Closer

An interactive visual experience exploring relationships and interactions within a space through real-time generative visuals. This project utilizes pose recognition to influence 3D environments.

Tools & Resources

  • TouchDesigner
  • OpenCV & TensorFlow (Pose Recognition via Webcam)
  • ParticlesGPU

Project Timeline

Weeks 3–4: Preproduction & Asset Creation

Finalized concept, gathered resources (TouchDesigner, TensorFlow), and researched reference imagery for moving particles and color palettes. Created storyboard drawings and camera input tests.

Weeks 5–7: Prototyping & Interaction

Built a basic prototype with color and objects. Refined and tested interactivity with initial user reactions.

Weeks 8–12: Midterm Critique & Visual Polishing

Presented the first draft for feedback. Polished visual effects, and finalized video clips.

Weeks 13–15: Final Submission

Finalized all project elements, prepared technical specs, and submitted the work to professional opportunities.
